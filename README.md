# QuickActions 快捷操作面板

> KOReader 快捷操作补丁 - 在顶部菜单栏增加可自定义的快捷操作标签页

---

## 安装

将 `2-quickactions.lua` 放入 `koreader/patches/` 目录，重启 KOReader。

卸载：删除该文件，可选删除 `koreader/settings/quickactions.lua`

---

## 核心概念：三层结构

```
┌─────────────────────────────────────────────────────────────┐
│  Layer 1: 动作库（所有可用动作）                            │
│  ├── 内置动作（系统预置，24个）                             │
│  └── 自定义动作（用户创建）                                 │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼ 添加/移除
┌─────────────────────────────────────────────────────────────┐
│  Layer 2: 按钮面板（显示在快捷操作标签页上的按钮）          │
│  └── 从动作库中选择要显示的动作，按顺序排列                 │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼ 过滤显示
┌─────────────────────────────────────────────────────────────┐
│  Layer 3: 界面过滤（不同界面显示不同按钮）                  │
│  ├── 通用：文件管理器 + 阅读器都显示                       │
│  ├── 文件管理器专用：仅文件管理器显示                      │
│  └── 阅读器专用：仅阅读器显示                              │
└─────────────────────────────────────────────────────────────┘
```

## 五种自定义动作类型

### 1. 文件夹
跳转到指定目录

**创建**：设置菜单 → 快捷操作 → 新建 → 文件夹 → 选择目录

### 2. 集合
打开收藏夹中的某个集合

**创建**：设置菜单 → 快捷操作 → 新建 → 集合 → 选择集合

### 3. 插件
调用已安装插件的功能，自动扫描 FM/Reader 中的插件实例

**创建**：设置菜单 → 快捷操作 → 新建 → 插件 → 选择插件

### 4. 系统动作
调用 Dispatcher 系统动作

**创建**：设置菜单 → 快捷操作 → 新建 → 系统动作 → 选择动作
- 无参数：直接选择
- 数值参数：SpinWidget 选择数值
- 字符串参数：输入文本
- 可配置参数：子菜单选择

### 5. 菜单录制
通过点击菜单项录制路径，一键执行复杂操作

**创建**：设置菜单 → 快捷操作 → 新建 → 录制菜单动作 → 在菜单中依次点击目标项 → 点击"完成录制"

## 完整使用流程

### 快速开始

初次使用可直接点击面板上的“+”号按钮进入到“添加按钮”列表，选择“全部添加”将所有内置动作一键添加到面板，然后打开主菜单--编辑按钮--显示标签，熟悉按钮功能，当内置动作不能满足需要时再添加自定义操作。

### 第一步：创建自定义动作

**入口**：
- 面板上的"新建快捷操作"按钮
- 设置菜单 → 快捷操作 → 新建

**操作**：
1. 选择动作类型（文件夹/集合/插件/系统动作/菜单录制）
2. 选择图标（网格浏览：内置200+nerdfont图标，自动扫描koreader/icons以及koreader系统图标目录下的svg/png格式图片，另支持浏览文件切换到任意目录查找图标）
3. 选择界面（通用/文件管理器/阅读器）
4  按需修改动作名称
5. 点击"保存"

**结果**：
- 动作加入"自定义动作"列表（动作库）
- 若开启"保存时自动添加到按钮"，自动出现在面板中
- 若未开启，需手动添加到面板

### 第二步：将动作添加到面板

**入口**：
- 面板上的"添加按钮"
- 设置菜单 → 编辑按钮 → 添加按钮

**操作**：
- 浏览动作列表（内置动作 + 自定义动作）
- 点击动作切换开关（✓ 表示已添加，可一键批量添加）
- 已添加的动作出现在面板中
- 最多 66 个按钮

**特殊开关**：
- 前光滑块：显示/隐藏
- 色温滑块：显示/隐藏（设备支持时）
- 显示滑块数值：显示/隐藏
- 全部添加/全部取消：批量操作

### 第三步：调整按钮外观

**入口**：设置菜单 → 编辑按钮

| 设置 | 操作 |
|------|------|
| 排列按钮 | 拖拽排序 |
| 按钮形状 | 圆形/圆角方形/无边框 |
| 按钮背景 | 透明/实色/浅灰 |
| 显示标签 | 开/关 |
| 按钮大小 | SpinWidget 选择 60%-150% |
| 标签大小 | SpinWidget 选择 50%-200% |

### 第四步：设置界面过滤

**入口**：设置菜单 → 界面过滤

**操作**：
1. 启用界面过滤（开关）
2. 进入"文件管理器专用"或"阅读器专用"
3. 点击动作切换专用状态
4. 或使用"全部专用/全部取消"批量操作

**效果**：
- 专用按钮仅在对应界面显示
- 通用按钮在所有界面显示
- 菜单动作自动锁定为录制时的界面

### 第五步：长按操作

#### 长按面板标题 → 打开设置菜单

在面板顶部标题区域长按（需开启"长按标签页面板打开设置"），直接打开设置菜单。

#### 长按按钮 → 编辑

在面板中长按任意按钮（需开启"长按按钮打开编辑框"）：

**内置动作长按**：
- 编辑名称
- 更换图标
- 更换界面
- 排序（左移/右移/位置跳转）
- 移除出面板

**自定义动作长按**：
- 编辑名称
- 更换图标
- 更换界面（菜单动作自动锁定）
- 更换动作类型和参数
- 排序（左移/右移/位置跳转）
- 移除出面板
- 删除自定义动作（从动作库中彻底删除）

#### 排序操作详解

长按按钮进入编辑界面后：

| 按钮 | 功能 |
|------|------|
| ◀ | 左移一位 |
| 位置数字（如 3/10） | 打开排序界面，自由拖拽排列 |
| ▶ | 右移一位 |
| 移除 | 仅从面板移除，动作保留在动作库 |
| 删除 | 仅自定义动作有，从动作库彻底删除 |

#### 移除 vs 删除

| 操作 | 影响范围 | 可恢复性 |
|------|----------|----------|
| 移除 | 仅从面板移除，动作库保留 | 可通过"添加按钮"重新加入 |
| 删除 | 从动作库彻底删除 | 需重新创建 |

## 图标系统

### 支持的图标类型

| 类型 | 格式示例 | 存放位置 |
|------|----------|----------|
| Nerd Font | `nerd:F067` | 无需存放，系统字体支持 |
| SVG 文件 | `my_icon.svg` | `koreader/icons/` |
| PNG 文件 | `my_icon.png` | `koreader/icons/` |

### 图标选择器

**入口**：编辑动作时点击"图标"按钮

**界面**：
- 网格预览，每页 7×5
- 支持两种分类："Nerd Font" 和 "文件图标"
- 支持按名称筛选（输入框）
- 支持"浏览文件"（PathChooser 浏览目录）
- 支持 Nerd Font 手动输入码位

### 系统图标替换

**入口**：设置菜单 → 系统图标替换

**操作**：
1. 打开后显示所有系统图标（网格预览）
2. 被替换的图标显示黑色边框
3. 点击任意图标，弹出选择界面
4. 选择新图标或"应用默认"恢复
5. 点击"应用替换"保存
6. 重启 KOReader 生效

**批量操作**：
- 重置全部：恢复所有系统图标为默认
- 应用替换：保存当前所有替换

## UI字体切换

**入口**：设置菜单 → UI字体切换

### 三种字体类型

| 类型 | 影响范围 | 默认字体 |
|------|----------|----------|
| 常规字体 | 菜单、对话框、通知等主要UI文本 | NotoSans-Regular.ttf |
| 粗体字体 | 标题、按钮等 | NotoSans-Bold.ttf |
| 等宽字体 | 输入框等 | DroidSansMono.ttf |

### 操作流程

1. 将字体文件（.ttf/.otf）放入 `koreader/fonts/` 目录
2. 打开"UI字体切换"
3. 点击要切换的字体类型
4. 从列表中选择字体
5. 重启 KOReader 生效

### 重置

- 点击"重置全部"恢复所有字体为默认
- 弹出确认框，可选择立即重启

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 面板不显示 | 检查"启用标签页"是否开启；删除配置文件重启 |
| Nerd Font 不显示 | 确认系统安装 Nerd Fonts 字体 |
| SVG/PNG 图标不显示 | 放入 `koreader/icons/` 目录 |
| 系统图标替换不生效 | 需重启 KOReader |
| 字体切换不生效 | 字体文件放入 `koreader/fonts/` 目录；重启 KOReader |
| 插件动作不可用 | 确认插件已安装并启用 |
| 菜单录制不执行 | 确认录制的路径在当前界面可访问 |

# 封面视觉设置：2-fm-cover

## 概述

- 统一处理书籍封面、文件夹封面，优化无封面书籍占位符
- 同时适用于Mosaic（网格封面）和 List（列表封面）两种显示模式）
- 兼容simpleui.koplugin（关闭simpleui书库相关设置）

![fmcover](picture/fmcover.jpg)

---

## 功能列表

![fmcover1](picture/fmcover1.jpg)

### 一、书籍封面

![fmcover2](picture/fmcover2.jpg)

#### 1. 占位封面

![fmcover3](picture/fmcover3.jpg)

当书籍没有内置封面时自动生成：

| 样式 | 效果 |
|------|------|
| 简单 | 纯白背景 + 黑色文字 |
| 渐变 | 蓝紫色渐变背景 + 深色文字 |

#### 2. 徽章系统

![fmcover4](picture/fmcover4.jpg)

| 徽章 | 位置 | 说明 |
|------|------|------|
| 收藏星标 | 左上角 | 标记已收藏书籍 |
| 阅读进度 | 右上角 | 显示 1%-99% 进度 |
| NEW 横幅 | 右上角斜角 | 标记新添加书籍 |
| 页面计数 | 左下角 | 显示总页数（K/M 格式化） |
| 格式徽章 | 右下角 | 显示 EPUB/PDF/CBZ 等格式 |

#### 3. 状态反馈
- **已读书籍变暗**：读完的书籍封面亮度降低至 60%

---

### 二、文件夹封面

![fmcover5](picture/fmcover5.jpg)

#### 1. 封面获取优先级
1. 显式封面文件（`cover.jpg`、`cover2.jpg`、`cover3.jpg`、`cover4.jpg`）
2. 文件夹内书籍封面（按 KOReader 排序规则取前 N 本）
3. 自动生成占位封面（显示文件夹名）

#### 2. 封面显示模式

![fmcover6](picture/fmcover6.jpg)

| 模式 | 说明 |
|------|------|
| Gallery | 4宫格拼贴，最多4个封面 |
| Stack | 堆叠效果，封面缩放72%后错位堆叠 |
| Normal | 只显示第一本书的封面（铺满） |
| None | 不显示封面，只显示文件夹名 |

#### 3. 文件夹封面叠加元素

| 元素 | 位置/样式 |
|------|----------|
| 文件夹名称 | 居中或底部，可半透明/不透明 |
| 文件数量徽章 | 右上角圆形徽章 |
| 书脊装饰线 | 左侧或顶部双线条装饰 |

## 使用说明

- 全部遮盖：点击勾选即可批量遮盖所有可遮盖样式的标注，取消勾选则批量揭开所有遮盖，还原原本样式。（可设置快捷手势：阅读器-cover all/uncover all）
- 单个遮盖-切换模式：默认双击可以切换单个标注的遮盖状态（但有些人可能设置了禁用双击，需要取消禁用双击才能实现切换）。可以更改为单击切换，但是单击标注本身会弹出koreader原始菜单，会有一些干扰，不介意这点干扰的话可以开启单击切换。
- 可遮盖样式：默认为高亮（即仅高亮应用遮盖模式），可更改，也可同时选择多个样式
- PS：遮盖模式仅对当前打开的书籍有效，不会影响原本的标注数据，关闭书籍后重新打开，仍旧是原本样式（即非遮盖模式）
