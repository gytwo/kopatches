# 快捷操作面板

> KOReader 快捷操作补丁 - 在顶部菜单栏增加可自定义的快捷操作标签页

---

## 安装

将 `2-quickactions.lua` 放入 `koreader/patches/` 目录，重启 KOReader。

> 卸载：删除该文件，可选删除 `koreader/settings/quickactions.lua`（配置文件）。

---

## 快速开始

初次使用？按以下步骤快速上手：

1. **打开面板**：点击顶部菜单栏的快捷操作标签页（⭐ 图标）

2. **体验默认按钮**：面板已预置 WiFi、夜间模式、旋转等常用功能，点击即可使用

3. **添加更多内置动作**：点击面板上的 **[添加按钮](#添加按钮)** → 点击 **☐ 全部添加** → 所有内置动作一键加入面板。

4. **显示按钮标签**：**[设置菜单](#设置菜单)** → **[编辑按钮](#编辑按钮)** → 开启 **显示标签**，方便识别按钮功能

5. **按需自定义**：如果内置动作不够用，点击 **[新建自定义动作](#新建自定义动作)** 创建属于自己的快捷动作。

## 核心概念：三层结构

```
┌─────────────────────────────────────────────────────────────┐
│ Layer 1: 动作库（所有可用动作） │
│ ├── 内置动作（系统预置，26个） │
│ └── 自定义动作（用户创建） │
└─────────────────────────────────────────────────────────────┘
│
▼ 添加/移除
┌─────────────────────────────────────────────────────────────┐
│ Layer 2: 按钮面板（显示在快捷操作标签页上的按钮） │
│ └── 从动作库中选择要显示的动作，按顺序排列 │
└─────────────────────────────────────────────────────────────┘
│
▼ 过滤显示
┌─────────────────────────────────────────────────────────────┐
│ Layer 3: 界面过滤（不同界面显示不同按钮） │
│ ├── 通用：文件管理器 + 阅读器都显示 │
│ ├── 文件管理器专用：仅文件管理器显示 │
│ └── 阅读器专用：仅阅读器显示 │
└─────────────────────────────────────────────────────────────┘

```

## 新建自定义动作

当[内置动作](#内置动作列表)不满足需求时，可通过以下方式创建自定义动作。

### 入口（任选其一）

- 点击面板上的 **新建快捷操作** 按钮
- 或通过 [设置菜单](#设置菜单) → 快捷操作 → 新建

### 操作步骤

> 💡 在弹出的[编辑对话框](#编辑对话框)对话中中进行编辑

1. **选择动作类型**（五种）

| 类型 | 说明 |
|------|------|
| **文件夹** | 跳转到指定目录 |
| **集合** | 打开收藏夹中的某个集合 |
| **插件** | 调用已安装插件的功能（自动扫描并列出） |
| **系统动作** | 调用 Dispatcher 系统动作（支持所有类型） |
| **菜单录制** | 点击菜单项录制路径，一键执行复杂操作 |

2. **选择图标**

> 💡 在 [图标选择器](#图标选择器)中选择图标

3. **选择界面过滤**

| 选项 | 说明 |
|------|------|
| **通用** | 文件管理器 + 阅读器均显示 |
| **文件管理器专用** | 仅在文件管理器显示 |
| **阅读器专用** | 仅在阅读器显示 |

> 💡 菜单动作自动锁定为录制时的界面，不可更改。详见 [界面过滤](#界面过滤)

4. **修改动作名称**（可选）

> 💡动作名称会根据选择的动作自动填充，用户可直接修改。

5. 点击 **保存**

> 💡 保存后，动作自动加入动作库（可在 [添加按钮](#添加按钮) 中查看）

- 若开启 **保存时自动添加到按钮**（默认开启），自动出现在面板中
- 若未开启，需通过 [添加按钮](#添加按钮) 手动加入面板

所有按钮（包括内置动作和自定义动作）均可通过长按进入编辑界面。

在面板中长按任意按钮（默认开启，可在设置中关闭）即可进入编辑界面：

## 界面过滤

设置按钮在不同界面下的可见性，让面板在不同场景下自动显示最相关的按钮。

### 入口

| 入口方式 | 说明 |
|----------|------|
| [设置菜单](#设置菜单) → **界面过滤** | 集中管理所有动作的界面过滤设置 |
| **新建自定义动作** → 选择界面过滤 | 创建动作时直接指定其界面过滤级别 |
| **长按按钮** → 编辑 → 界面 | 修改已有动作的界面过滤级别 |

> 💡 以上任一方式均可设置界面过滤。

### 启用界面过滤

开启后，按钮将根据其界面设置在对应界面显示或隐藏。关闭则所有按钮在所有界面都显示。

### 三种界面过滤级别

| 级别 | 显示位置 | 适用场景 |
|------|----------|----------|
| **通用** | 文件管理器 + 阅读器均显示 | 功能与当前操作界面无关，任何场景都能用 |
| **文件管理器专用** | 仅在文件管理器显示 | 仅与文件管理相关的功能，如封面视觉设置 |
| **阅读器专用** | 仅在阅读器显示 | 仅与阅读相关的功能，如字体列表、遮盖模式 |

### 专用列表管理

点击 **文件管理器专用** 或 **阅读器专用** 进入专用列表：

| 操作 | 说明 |
|------|------|
| 点击动作切换专用状态 | ✓ 表示该动作仅在当前界面显示 |
| **全部专用** | 一键将当前界面下所有可手动设置界面的动作设为专用（菜单录制类动作除外） |
| **全部取消** | 一键取消当前界面下所有可手动设置界面的动作的专用状态（菜单录制类动作除外）  |

> 💡 菜单录制类动作的界面在录制时自动锁定，不可更改，因此不参与批量操作。

### 重置专用

点击 **重置为默认专用** 恢复所有动作为默认状态：

| 动作类型 | 重置结果 |
|----------|----------|
| 内置动作 | 恢复为代码中定义的默认 view |
| 自定义动作 | 恢复为创建时选择的 view |
| 菜单动作 | 保持录制时的界面锁定，不受重置影响 |

## 图标选择器

QuickActions 内置统一的图标选择界面，用于为动作、标签页等选择图标。

- 网格浏览内置 200+ Nerd Font 图标
- 自动扫描 `koreader/icons/` 及系统图标目录下的 SVG/PNG
- 支持按 **图标名称** 或 **Nerd Font 码位** 搜索筛选
- 支持浏览文件切换到任意目录查找图标
- 支持手动输入 Nerd Font 十六进制码位添加图标

| 模式 | 支持类型 | 触发入口 |
|------|----------|----------|
| **完整模式** | Nerd Font + SVG/PNG 文件 | 编辑动作图标、[新建自定义动作](#新建自定义动作) |
| **文件模式** | 仅 SVG/PNG 文件（不支持 Nerd Font） | [设置菜单](#设置菜单) → 标签页图标、[系统图标替换](#系统图标替换) |

> 💡 **标签页图标** 和 **系统图标替换** 仅支持 SVG/PNG 格式文件，不支持 Nerd Font 图标。

## 系统图标替换

可以全局替换 KOReader 内置的系统图标（如菜单图标、按钮图标等）。

> ⚠️ **注意**：仅支持 SVG/PNG 格式，不支持 Nerd Font 图标。详见 [图标选择器](#图标选择器)。

### 入口

[设置菜单](#设置菜单) → **系统图标替换**

### 操作步骤

1. 进入后以网格形式展示所有系统图标，被替换的图标显示黑色边框
2. 点击任意图标，弹出选择界面（详见 [图标选择器](#图标选择器)）
3. 从图标库中选择新图标，或点击 **应用默认** 恢复为系统默认
4. 点击 **应用替换** 保存所有更改
5. 重启 KOReader 生效

### 批量操作

| 按钮 | 功能 |
|------|------|
| **重置全部** | 恢复所有系统图标为默认（需重启生效） |
| **应用替换** | 保存当前所有替换（需重启生效） |

> 💡 替换仅在当前设备生效，如需多设备同步请同步配置文件 `koreader/settings/quickactions.lua`。

## UI字体切换

切换 KOReader 界面字体（菜单、对话框、按钮等所有 UI 文字），不影响阅读器内书本内容的字体。

### 入口（任选其一）

- 点击面板上的 **切换UI字体** 按钮（需通过 [添加按钮](#添加按钮) 加入面板）
- [设置菜单](#设置菜单) → **UI字体切换**

### 三种字体类型

| 类型 | 影响范围 | 默认字体 |
|------|----------|----------|
| **常规字体** | 菜单、对话框、通知等主要 UI 文本 | NotoSans-Regular.ttf |
| **粗体字体** | 标题、按钮等 | NotoSans-Bold.ttf |
| **等宽字体** | 输入框等 | DroidSansMono.ttf |

### 操作步骤

1. 打开 **UI字体切换**
2. 点击要切换的字体类型（常规/粗体/等宽）
3. 从列表中自动列出设备中所有可用字体，选择目标字体
4. 重启 KOReader 生效

> 💡 如设备中已安装的字体不满足需求，可自行将字体文件（`.ttf` / `.otf`）放入 `koreader/fonts/` 目录，重启后即可在列表中看到。

### 重置

点击 **重置全部** 恢复所有字体为默认 → 弹出确认框 → 点击 **重启** 对话框字体立即生效，标签页菜单字体需重启后才生效。

### 入口（任选其一）

- 如未添加，通过 [添加按钮](#添加按钮) 加入面板

### 操作步骤

1. 打开一本书
2. 点击面板上的 **字体列表** 按钮

5. 当前使用的字体旁显示 ✓ 标记

> 💡 如设备中已安装的字体不满足需求，可自行将字体文件（`.ttf` / `.otf`）放入 `koreader/fonts/` 目录，重启后即可在列表中看到。

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 面板不显示 | 检查"启用标签页"是否开启；删除配置文件`koreader/settings/quickactions.lua`重启 |
| Nerd Font 不显示 | 如果有某些图标显示为问号图标，可能是因为版本对应的码不适配，可以直接更新本补丁文件中硬编码的Nerd Font码或者删除问题编码 |
| SVG/PNG 图标不显示 | 将图标文件放入 `koreader/icons/` 目录 |
| UI字体切换不生效 | 标签页菜单字体需要重启KOReader才生效 |
| 联动插件/补丁动作不可用 | 确认插件已安装并启用，如果还是不行可能是因为原插件/补丁已更新，对应的动作接口发生变化，需修改注册代码 |
| 菜单录制不执行 | 可能是菜单结构变化（如系统更新、安装/更新其他插件补丁注入或修改了菜单列表）导致原录制时的菜单路径无法正确指向当前菜单，需要重新录制动作 |

## 使用说明

- 全部遮盖：点击勾选即可批量遮盖所有可遮盖样式的标注，取消勾选则批量揭开所有遮盖，还原原本样式。（可设置快捷手势：阅读器-cover all/uncover all）
- 单个遮盖-切换模式：默认双击可以切换单个标注的遮盖状态（但有些人可能设置了禁用双击，需要取消禁用双击才能实现切换）。可以更改为单击切换，但是单击标注本身会弹出koreader原始菜单，会有一些干扰，不介意这点干扰的话可以开启单击切换。
- 可遮盖样式：默认为高亮（即仅高亮应用遮盖模式），可更改，也可同时选择多个样式
- PS：遮盖模式仅对当前打开的书籍有效，不会影响原本的标注数据，关闭书籍后重新打开，仍旧是原本样式（即非遮盖模式）
